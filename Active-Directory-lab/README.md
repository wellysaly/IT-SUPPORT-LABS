This lab focused on basic Active Directory administration in a Windows domain environment. The goal was to understand how user accounts and access permissions are managed in an organization.

Tools Used
Windows Server
Active Directory Users and Computers
VirtualBox

Problem Solved
Simulated common helpdesk scenarios such as users being unable to log in, forgotten passwords, and incorrect access permissions. Resolved these issues by resetting passwords, unlocking accounts, assigning users to groups, and verifying access rights.
